# TaxiNYProjectDE
Data Engineering NYC Taxi Project, from raw data, ETL on MAGE and Google VM to BigQuery and data Visualization on Looker.
DataBase From: https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page
